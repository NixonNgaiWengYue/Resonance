# Resonance

## Setup Instructions

### 1. Clone the repo

```bash
git clone https://github.com/NixonNgaiWengYue/Resonance.git
cd resonance
```

### 2. Download the database from Hugging Face

```bash
pip install -r requirements.txt
python3 download_data.py
```

### 3. Start Backend (Flask API)

```bash
python3 server.py
```

### 4.Start Frontend (React)
```bash
npm install
npm run dev
```
